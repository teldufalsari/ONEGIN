◆ main()

int main ( )

Main function ins used to control other functions.

Parameters
*textThe main array where the text is stored.
name[MAXNAME]Buffer where the name of the opened file is stored
sizeSize of the opened file in bytes.
n_strNumber of strings in the opened file.
indexArray of pointers to strings of given file.
new_name[MAXNAME]Name/address of the output file.

The algorythm of its work is:

  • to load input file into array;
  • to generate an array **index of pointers to strings in this array;
  • to arrange pointers in **index so that pointed lines would be put in desired order;
  • to create new text file and to write sorted lines into it.
